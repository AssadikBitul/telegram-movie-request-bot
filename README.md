# Telegram Movie Request Bot (Render Deployment)

A simple Telegram bot that receives movie requests from users and forwards them to the admin.

## Features
✔️ User sends movie name  
✔️ Bot forwards to Admin  
✔️ Admin sees request instantly  
✔️ 24/7 Online using Render.com  
✔️ GitHub Auto Deploy  

## Deploy on Render
1. Create a free account → https://render.com
2. Click "New" → "Background Worker"
3. Connect your GitHub repo
4. Add Environment Variables:

BOT_TOKEN = YOUR_BOT_TOKEN  
ADMIN_ID = YOUR_TELEGRAM_ID

5. Click Deploy → Done!
